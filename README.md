# Sistema de Turnos y Compensatorios

## Archivos
- `app.py` — aplicación principal (Streamlit / Flask)
- `index.html` — interfaz completa del sistema
- `requirements.txt` — dependencias

## Correr localmente

### Con Streamlit (recomendado)
```bash
pip install streamlit
streamlit run app.py
```

### Con Flask
```bash
pip install flask
python app.py
```

## Despliegue en Streamlit Cloud
1. Sube los tres archivos a un repositorio de GitHub
2. Ve a [share.streamlit.io](https://share.streamlit.io)
3. Conecta el repositorio
4. El archivo principal es `app.py`
